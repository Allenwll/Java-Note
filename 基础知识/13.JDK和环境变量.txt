JDK的介绍
	JDK 
		Java Development Kit Java标准版开发包
	JRE
		是Java运行时环境（Java Runtime Enviroment）的简称
		是运行Java程序必需的条件
		
	JVM
		是Java虚拟机（Java Virtual Machine）的简称
		
	JAR
		Java Archive，Java归档文件，是与平台无关的文件格式，它允许将许多文件组合成一个压缩文件
	
	API 
		Application Programming Interface 应用程序编程接口的缩写 是一些预定义函数  核心的类库

环境变量
	是一个具有特定名字的对象，它包含了一个或者多个应用程序将使用到的信息
	可以在任意路径下调用
	
	系统变量
		JAVA_HOME :C:Program Files\Java\jdk1.7.0_25 JDK的根目录 总路径
	用户变量
		Path：JDK根目录下的bin（二进制）or %JAVA_HOME%\bin 命令路径
		Class_Path:%JAVA_HOME%\lib 加载类路径
